| שם עמודה          | תיאור                                                      |
| :---------------- | :--------------------------------------------------------- |
| `id`              | מזהה שורה ייחודי.                                          |
| `hierarchy`       | מיקום היישוב בהיררכיה המנהלתית (במידה וצוין).              |
| `galil`           | הגליל / האזור הגיאוגרפי-היסטורי הרחב (למשל: Rus).          |
| `kahal`           | שם עיר הקהל הראשי שאליה שויך היישוב.                       |
| `name`            | שם היישוב כפי שהופיע ברשומות המפקד ההיסטורי.               |
| `settlement_type` | סוג היישוב (למשל: city, village).                          |
| `pop`             | גודל האוכלוסייה היהודית ביישוב באותה שנת מפקד.             |
| `year`            | שנת המפקד (לרוב 1765, אך עשוי להיות 1778 או 1784 בהשלמות). |
| `region`          | קוד אזור היסטורי/מחקרי.                                    |
| `modern_name`     | השם המודרני של היישוב (לרוב באוקראינה או פולין של ימינו).  |
| `viv`             | מחוז / פלך היסטורי (Voivodeship, למשל Bracław).            |
| `pow`             | נפה / פוביאט (Powiat).                                     |
| `geocode`         | קוד מזהה גיאוגרפי ייחודי המשויך למקום.                     |
| `lat`             | קו רוחב (Latitude) למיקום המודרני של היישוב.               |
| `lon`             | קו אורך (Longitude) למיקום המודרני של היישוב.              |
| `kahal_id`        | קוד מזהה מספרי עבור עיר הקהל.                              |
| `subunit`         | תת-יחידה מנהלתית בתוך הקהל (אם רלוונטי).                   |
