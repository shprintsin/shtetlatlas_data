| שם משתנה | תיאור |
| :--- | :--- |
| `line_id` | מספר רץ, מזהה ייחודי לכל היישובים בקובץ |
| `code` | מזהה קהילה מורכב (שם הפרובינציה, המחוז, ו-4 אותיות ראשונות של שם היישוב ללא עיצורים) |
| `jg_id` | מזהה היישוב במערכת JewishGen |
| `name` | שם היישוב |
| `enc_name` | שם היישוב כפי שמופיע באנציקלופדיה |
| `modern_name` | השם המודרני של היישוב |
| `prov` / `dist` | פרובינציה / מחוז |
| `area` | אזור גיאוגרפי היסטורי (למשל: Mainland Russia, Galicia, Pale, Prussia, Bessarabia, Bukovina) |
| `presence_type` | סגנון העדות לקיום הקהילה ( `community_founded`, `first_mention`, `jews_arrived`, `jews_present`) |
| `year_code` | קוד תקופה להערכת שנת ההקמה (`XX` - טווח מאה, `EX` - תחילת מאה, `LX` - סוף מאה, `MX` - אמצע מאה) |
| `lower_bound` | השנה הראשונה בה ייתכן שקיימת קהילה ביישוב |
| `upper_bound` | השנה האחרונה עבורה בוודאות קיימת קהילה ביישוב |
| `mid_range` | אמצע טווח בין שנות ההערכה (מעוגל ל-5 שנים) |
| `ziv_yearcode` | קוד תקופה לפי מתודולוגיה קודמת |
| `note` | הערות נוספות |
| `lon` / `lat` | קווי אורך ורוחב (מערכת ייחוס: EPSG:4326) |
| `geom_source` | המקור לקואורדינטות הגיאוגרפיות |
| `enc_page` | עמוד הערך הרלוונטי באנציקלופדיה |
| `entry_source` | מקור/מחבר הערך (ZIV, SHN) |
| `osm_id` | מזהה היישוב במערכת OpenStreetMap |
| `wikidata_id` | מזהה היישוב בפרויקט Wikidata |
| `wikipedia_url` | קישור לערך בוויקיפדיה אודות היישוב |
