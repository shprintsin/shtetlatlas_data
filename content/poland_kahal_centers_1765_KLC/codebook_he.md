| שם משתנה                        | סוג    | תיאור                                                        |
| :------------------------------ | :----- | :----------------------------------------------------------- |
| **מזהי רשומה (Metadata)**       |        |                                                              |
| `id`                            | מספר   | מזהה שורה ייחודי במסד הנתונים.                               |
| `geocode`                       | מחרוזת | קוד מזהה גיאוגרפי ייחודי                                     |
| `sum_group`                     | קטגורי | קוד קבוצת סיכום סטטיסטית לאימות.                             |
| `group`                         | מחרוזת | שיוך קבוצתי או אזורי.                                        |
| **שמות (Names)**                |        |                                                              |
| `name`                          | מחרוזת | שם הקהל.                                                     |
| `name_he`                       | מחרוזת | שם עברי (עבור קהילות ידועות).                                |
| `modern_name`                   | מחרוזת | השם והמיקום המודרני של יישוב הקהל (לרוב באנגלית).            |
| **שיוך מנהלי**                  |        |                                                              |
| `viv`                           | קטגורי | פלך / מחוז היסטורי (Voivodeship).                            |
| `viv_code`                      | קטגורי | קוד מזהה של הפלך.                                            |
| `powiat`                        | קטגורי | נפה.                                                         |
| `galil`                         | קטגורי | שם הגליל (מחוז קהל) למשל: Great Poland).                     |
| `galil_id`                      | מספר   | מזהה מספרי של הגליל.                                         |
| **דמוגרפיה ואוכלוסייה**         |        |                                                              |
| `jewish_pop`                    | מספר   | סך כל האוכלוסייה בקהל (כולל התוספות המאוחרות, במידה והיו).   |
| `addition`                      | שלם    | כמות האוכלוסייה/משלמי המס שנוספה לחישוב המקורי.              |
| `addition_type`                 | מחרוזת | תיאור מהות התוספת (למשל: תוספת כפרים, חובות עבר).            |
| **היררכיה וטקסט**               |        |                                                              |
| `is_child`                      | בינארי | מציין האם הקהל מוגדר כתת-קהילה.                              |
| `parent_id`                     | מספר   | במידה ומדובר בתת-קהילה, מזהה ה-id של קהל האם אליו היא כפופה. |
| `text`                          | מחרוזת | הטקסט המקורי כפי שהופיע ברשומות המפקד ההיסטוריות.            |
| **קואורדינטות ומזהים חיצוניים** |        |                                                              |
| `lat`                           | מספר   | קו רוחב (Latitude) של המיקום המודרני.                        |
| `lon`                           | מספר   | קו אורך (Longitude) של המיקום המודרני.                       |
| `wikidata`                      | מחרוזת | מזהה הפריט במסד הנתונים של ויקינתונים (Wikidata).            |
| `wikipedia`                     | מחרוזת | שם הערך המקושר בוויקיפדיה.                                   |
| `osm_id`                        | מספר   | מזהה המיקום במפות OpenStreetMap.                             |
