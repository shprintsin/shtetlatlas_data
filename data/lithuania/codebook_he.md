
**מבנה הנתונים (Codebook)**
המאגר מורכב משני קבצים מרכזיים:

**1. קובץ הנתונים המפורט: `lithuanian_kahal_detailed_1765.csv`**
קובץ זה מכיל את הפירוט המלא של היישובים השונים תחת כל קהל.

| שם עמודה | תיאור |
| :--- | :--- |
| `volume` | מספר הכרך בארכיון LVIA ממנו נלקח המידע. |
| `kahal_name` | שם הקהל הראשי שאליו השתייך היישוב. |
| `settlement_name` | שם היישוב או הכפר. |
| `settlement_type` | סוג היישוב כפי שהוגדר במפקד: עיר (miasto), עיירה (miasteczko), כפר (wieś), או בית מרזח (karczma). |
| `pop` | גודל האוכלוסייה היהודית ביישוב בשנת 1765. |


**2. קובץ הסיכום: `lithuanian_kahal_summary_1765.csv`**
קובץ המרכז מידע אודות 83 הקהלים המופיעים בקובץ המפורט, ומאפשר מבט מאקרו על פריסת האוכלוסייה.

| שם עמודה | תיאור |
| :--- | :--- |
| `kahal_id` | מזהה ייחודי לקהל. |
| `name` | שם הקהל. |
| `settlement_type` | סוג היישוב המרכזי של הקהל (לרוב miasto). |
| `pop` | סך כל האוכלוסייה בקהל (העיר + סביבותיה). |
| `miasto_pop` | גודל האוכלוסייה היהודית בעיר המרכזית בלבד. |
| `surroundings_count` | מספר היישובים/הכפרים בסביבת העיר השייכים לקהל. |
| `surroundings_pop` | סך האוכלוסייה היהודית ביישובים הסובבים. |
| `year` | שנת המפקד (1765). |
| `volume` | מספר הכרך בארכיון. |
| `powiat` | המחוז המנהלתי (למשל Kowieński, Żmudzkie). |
| `viv_code` | קוד אזור היסטורי. |
| `geocode` | קוד גיאוגרפי לזיהוי. |
