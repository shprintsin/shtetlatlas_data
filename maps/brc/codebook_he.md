# ספר קודים — מפת בראצלב

## שכבת מחוזות

| שדה | תיאור |
|---|---|
| kahal | שם הקהל כפי שהופיע ברשימות סיכום המפקד |
| galil | שם הגליל (הקהל המחוזי) בו נמצא הקהל |
| historical_name | שם היסטורי של היישוב |
| settlement_type | סוג היישוב (Miasto = עיר, Miasteczko = עיירה, Wieś = כפר) |
| jewish_pop | אוכלוסייה יהודית ביישוב הראשי |
| year | שנת המפקד (1765) |
| region | קוד אזור (BRT = בראצלב) |
| geocode | קוד גיאוגרפי ייחודי בפורמט {region}_{kahal_name} |
| source | מזהה מקור הנתונים (YGZ) |
| kahal_id | מזהה מספרי ייחודי של הקהל |
| he_name | שם הקהל בעברית |
| viv | וויוודשיפ (Województwo) — יחידה מנהלית ראשונה ב-1765 (נפה) |
| pow | פוביאט (Powiat) — יחידה מנהלית שנייה ב-1765 (מחוז) |
| rural | מספר יהודים שגרו בכפרים, בפונדקים ובאחוזות שמסביב |
| urban | מספר יהודים שגרו ביישוב הראשי |
| total | סך כל האוכלוסייה היהודית (עירונית + כפרית) |
| places | מספר היישובים הכפריים שתחת סמכות הקהל |

## שכבת נקודות

| שדה | תיאור |
|---|---|
| id | מזהה שורה ייחודי |

| galil | שם הגליל (הקהל המחוזי) |
| kahal | שם הקהל אליו משתייך היישוב |
| name | שם היסטורי של היישוב כפי שנרשם במפקד |
| settlement_type | סוג היישוב (city, town, village, estate וכו') |
| pop | אוכלוסייה יהודית ביישוב |
| year | שנת המפקד (1765) |
| region | קוד אזור (BRT = בראצלב) |
| modern_name | שם מודרני של היישוב, ניתן לחיפוש בגוגל מפות |
| viv | וויוודשיפ (Województwo) — יחידה מנהלית ראשונה ב-1765 (נפה) |
| pow | פוביאט (Powiat) — יחידה מנהלית שנייה ב-1765 (מחוז) |
| geocode | קוד גיאוגרפי ייחודי בפורמט {region}_{name} |
| kahal_id | מזהה מספרי של הקהל אליו משתייך היישוב |
| subunit | יחידת משנה, אם רלוונטי |
